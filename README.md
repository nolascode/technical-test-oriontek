
# Technical Test

This project is part of a recruitment process as a Software Developer by the company Oriontek.
## Stack

 - [Net Core 6.0](https://learn.microsoft.com/es-es/aspnet/core/?view=aspnetcore-6.0)
 - [Dapper](https://github.com/DapperLib/Dapper)
 - [Mysql](https://www.mysql.com/)
 - [ReactJs](https://react.dev/)
 - [Redux Toolkit ](https://redux-toolkit.js.org/tutorials/overview)
 - [Tailwind CSS](https://tailwindcss.com/docs/installation)
 - [Swagger](https://swagger.io/)
